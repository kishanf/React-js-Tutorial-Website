What is React.js

React.js or ReactJS is a front end JavaScript library for building user interfaces. It was created by Facebook and was first used on their site in 2011 however it did not come into open source until mid 2013; Instagram.com is written entirley through the React JavaScript libraries. React.js essentially allows developers to create large web based applications with speed, simplicity and scalabilty. it is currently amoung the most popular JavaScript libraries available; with a large community of developers backing it. It uses Declarative views so that your code is more predictable and easier to debug, it encourages the creation of reusable code and a reuseable UI Componets. React.js has multiple big coperate users such as Netflix, Airbnb and PayPal; according to the live statistics you can find on Libscore.com.
